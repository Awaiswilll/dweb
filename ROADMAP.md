# dweb OS — Roadmap

> **One install. Every stack. Your own internet.**

This roadmap tracks the development of dweb OS. Items are organized by priority and timeline.

---

## 🚀 v0.1.0 — Initial Release (Current)

✅ **Released** — June 28, 2026

- Dev Portal Dashboard (React 19 + TypeScript)
- P2P Networking (HyperDHT + WebRTC)
- AI Build Agent (15+ providers, 100+ free models)
- .dweb Domain System
- Git Integration (GitHub OAuth)
- Cloud Deployment (AWS, Netlify, Vercel)
- WSL Distro (Alpine Linux)
- Windows Desktop App (Tauri v2)
- Microsoft Store name reserved
- Open source infrastructure (LICENSE, CODE_OF_CONDUCT, SECURITY.md, CONTRIBUTING.md)
