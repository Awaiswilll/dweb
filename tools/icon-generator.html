📋 How to Use These Icons for Microsoft Store

  1. Click "Download All Icons" above to save all PNG files
  2. Go to Partner Center → Select your "dweb" app
  3. Go to App icons section in the store listing
  4. Upload each icon to its corresponding slot:
    • StoreLogo.png → 50×50
    • Square44x44Logo.png → 44×44
    • Square71x71Logo.png → 71×71
    • Square150x150Logo.png → 150×150
    • Wide310x150Logo.png → 310×150
    • Square310x310Logo.png → 310×310
    • LargeTile.png → 284×284
    • SplashScreen.png → 620×300
  5. For Tauri app icons, replace files in src-tauri/icons/ with the generated sizes
